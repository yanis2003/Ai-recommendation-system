#  Système de Recommandation (Machine Learning)

Projet de M1 IA . Faire un poster en " Système de recommandation" 
Projet personelle . Système de recommandation inspiré de Netflix/Amazon.

##  Objectifs
- Manipuler un dataset utilisateur-produits
- Utiliser les similarités (cosine / corrélation)
- Filtrer utilisateurs & items
- Implémenter un algorithme simple de recommandation
- Visualiser les résultats

## Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Contenu
- Prétraitement des données
- Matrice utilisateurs-produits
- Recommandation basée sur la similarité
- Évaluation (RMSE)
